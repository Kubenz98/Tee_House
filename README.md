# Tee House

An online shop (Single Page Application), featuring Incremental Static Regeneration (ISR) called by Strapi webhook. Created using Next.js, React Query, Tailwind and Strapi.



## Tech Stack

- Next.js
- Strapi
- React Query
- Tailwind
